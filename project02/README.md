# Project 02: Responsive Navigation & Product Cards

## Objective

Update your navigation bar to be responsive, create a product card grid, and add click
functionality.

Requirements

1. Responsive Navigation Bar
- pdate your previous navigation bar to work on mobile and desktop
- Desktop: Horizontal layout (keep your flexbox styling)
- Mobile (< 768px): Stack items vertically
- Use a media query to handle the change
2. Product Card Grid
Create at least 5 product cards with:
- Product image
- Product name
- Brief description
- Price

Grid Layout:
- Use CSS Grid with 3 cards per row on desktop
- Responsive breakpoints:
  - Mobile (< 768px): 1 card per row
  - Tablet (768px - 1024px): 2 cards per row
  - Desktop (> 1024px): 3 cards per row
Styling:
- Add spacing between cards using gap
- Include borders or shadows on cards
- Add hover effects
3. JavaScript Click Event
- Add click event to each card
- Show an alert with the product name when clicked
- Example: Clicking "Nike Sneakers" shows alert("Nike Sneakers")
